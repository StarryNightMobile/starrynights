# Starry Mobile – Full Store  
Complete e‑commerce storefront with:
- Netlify Functions (Stripe, Paynow, Orders)
- Admin Panel
- CSV Export to GitHub
- SendGrid Emails
- Dark/Light Mode
- Fully Client-side Cart
- Product Pages

## Folders
- `netlify/functions/` — serverless back‑end  
- `details/` — product detail pages  
- `scss/` — SCSS source  
- `ts/` — TypeScript version of scripts  
- `react-version/` — starter React/Next.js migration folder

## Deployment
1. Push to GitHub  
2. Connect to Netlify  
3. Add environment variables  
4. Deploy
